# Hospital-Data-Health-Sector-Data-And - Study Case

## Objective
            This  project  mostly concenrates on Hospital Data And Study Case using sample Hospital data and
understand core concepts such as data loading, data transformation, data modeling, visualization, and reporting.


## Dataset
* Excel  Hospital data

## Dashboard Features
* load data into power bi 
* clean data in power query 
* Create relations between the table of data models in modeling
* Create measures like total patients,total Revenue,Success Rate,Total Appointments,etc... by using DAX functions 
* Design dashboards and reports in  visualization

# Tools Used
* Power BI
* Power Query
* Data modeling
* Dax
*visualization
